# filehost
A lightweight file hosting website.

## Features:
* Simple file upload interface.
* Duplicate Detection using SHA-256 Hashes.

## Upcoming Features:
* Expire files after 24 hours.
* Multiple files at once.
* UserTokens - Seeing past uploads.
* Admin panel for database management.

## Environment
Running on Artix Linux with the following packages: 
* nginx
* mariadb
* php
* php-fpm

### Configure:
* php.ini
* nginx.conf
